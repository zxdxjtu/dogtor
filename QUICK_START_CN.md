# 快速上手指南

## 🚀 3分钟快速开始

欢迎使用颈椎治疗助手！本指南将帮助您立即安装和开始使用扩展程序。

## 📦 安装步骤

### 方法1：从源码加载（推荐用于开发）

1. **下载扩展程序**
   ```bash
   git clone https://github.com/your-username/neck-therapy-extension.git
   cd neck-therapy-extension
   ```

2. **打开Chrome扩展页面**
   - 在地址栏输入 `chrome://extensions/`
   - 或者通过 Chrome菜单 → 更多工具 → 扩展程序

3. **启用开发者模式**
   - 在右上角切换"开发者模式"开关

4. **加载扩展程序**
   - 点击"加载已解压的扩展程序"
   - 选择 `neck-therapy-extension` 文件夹
   - 扩展图标应该出现在工具栏中

### 方法2：Chrome网上应用店（即将推出）

- 访问Chrome网上应用店
- 搜索"颈椎治疗助手"
- 点击"添加至Chrome"

## ⚡ 首次使用

### 1. 激活扩展程序

1. **点击工具栏中的扩展图标**
2. **切换开关**以启用自动旋转
3. **查看状态** - 您会看到"下次提醒：X分钟后"

### 2. 体验首次旋转

- 等待倒计时归零（或设置更短的间隔进行测试）
- 观察网页轻柔旋转：
  - **左旋转**（15°）持续30秒
  - **右旋转**（15°）持续30秒
  - **恢复正常**位置
- 跟随旋转移动头部来锻炼颈椎

### 3. 监控进度

- **状态指示器**：页面右上角的小指示器
- **倒计时器**：显示距离下次旋转的时间
- **阶段显示**：显示当前旋转阶段（左/右/正常）

## ⚙️ 基础设置

### 访问设置

1. 点击扩展图标
2. 点击齿轮（⚙️）图标打开选项
3. 或右键点击扩展图标 → "选项"

### 关键设置

| 设置项 | 默认值 | 范围 | 说明 |
|--------|--------|------|------|
| **旋转频率** | 10分钟 | 1-60分钟 | 旋转发生的频率 |
| **旋转时长** | 30秒 | 10-120秒 | 每次旋转持续时间 |
| **旋转角度** | 15度 | 5-45度 | 页面旋转的角度 |
| **显示指示器** | 开启 | 开启/关闭 | 显示状态指示器 |

### 快速调整

- **测试用途**：设置频率为1分钟
- **密集工作**：设置频率为30-60分钟
- **敏感用户**：减少角度至10度
- **隐蔽使用**：关闭状态指示器

## 🎯 使用技巧

### 最佳实践

1. **循序渐进**
   - 从默认设置开始（10分钟频率，15°角度）
   - 根据舒适度和工作模式调整

2. **跟随旋转**
   - 移动头部跟随屏幕旋转
   - 这样才能获得颈椎锻炼的效果
   - 不要抗拒旋转 - 配合它

3. **时机考虑**
   - 在长时间工作期间启用
   - 在视频通话或演示时禁用
   - 在密集计算机工作时使用较短间隔

### 智能功能

- **自动暂停**：在以下情况自动暂停：
  - 文本输入（在表单中打字）
  - 视频播放
  - 用户与页面元素交互

- **跨标签页同步**：设置和状态在所有浏览器标签页间同步

- **记忆功能**：在浏览器会话间记住您的偏好

## 🔧 故障排除

### 常见问题

**扩展在某些页面上不工作？**
- 某些特殊页面（chrome://、扩展页面）受到限制
- 这是正常的浏览器安全行为

**旋转感觉太突然？**
- 旋转使用平滑的CSS过渡
- 如果感觉突兀，尝试减少旋转角度

**想要临时暂停？**
- 点击扩展图标并关闭开关
- 或者开始打字 - 它会自动暂停

**设置没有保存？**
- 确保在选项页面点击"保存"
- 检查Chrome是否有存储数据的权限

### 重置为默认值

1. 打开选项页面
2. 点击"重置为默认值"
3. 确认重置
4. 所有设置返回原始值

## 📊 理解状态

### 状态指示器状态

- **绿点**：扩展激活，等待下次旋转
- **蓝点**：旋转进行中
- **橙点**：已暂停（检测到用户交互）
- **红点**：错误或不兼容页面
- **无点**：扩展已禁用

### 弹窗信息

- **下次提醒**：距离下次旋转的倒计时
- **今日总计**：已完成的旋转次数
- **会话时间**：扩展激活的时长
- **当前状态**：激活/暂停/禁用
